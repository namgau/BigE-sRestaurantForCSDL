Bước 1: Mở Terminal (CMD hoặc PowerShell), di chuyển vào thư mục dự án:

    cd "đường dẫn"

Bước 2: Tạo Virtual Environment (đã tạo sẵn thư mục venv):

    py -m venv venv

Bước 3: Kích hoạt Virtual Environment:

    [Windows CMD]
    venv\Scripts\activate.bat


    Khi kích hoạt thành công, đầu dòng lệnh sẽ hiện (venv).

Bước 4: Cài đặt thư viện cần thiết:

    pip install pyodbc redis PyQt6

Bước 5: Tắt Virtual Environment (khi không cần dùng nữa):

    deactivate


Mở file app_config.py và chỉnh sửa:

- DRIVER:  '{ODBC Driver 17 for SQL Server}' (hoặc bản mới hơn)
- SERVER:  Tên máy tính\SQLEXPRESS (ví dụ: localhost\SQLEXPRESS)
- DATABASE: 'RestaurantDB'
- Nếu dùng Windows Auth: giữ TRUSTED_CONNECTION = 'yes'
- Nếu dùng SQL Auth: đặt TRUSTED_CONNECTION = 'no', điền UID và PWD

KHỞI TẠO DATABASE (CHẠY MỘT LẦN DUY NHẤT)
-------------------------------------------------
Sau khi cấu hình app_config.py, chạy lệnh sau để tự động tạo
toàn bộ database (bảng, dữ liệu mẫu, trigger, stored procedure):

    python setup_database.py

Output thành công sẽ hiển thị:
    [✓] Kết nối SQL Server thành công!
    [✓] Hoàn tất! Đã thực thi X batch thành công.
    [✓] Database 'RestaurantDB' đã sẵn sàng:
        • 15 bảng: Bill, Booking, Category, ...
        • 3 trigger: trg_Bill_AutoSetPaidAt, ...
        • 3 stored procedure: sp_CancelBooking, ...

Lưu ý:
    - Script này an toàn khi chạy nhiều lần (không xóa dữ liệu cũ).
    - Nếu lỗi kết nối: kiểm tra SQL Server có đang chạy không
      và SERVER trong app_config.py có đúng không.



VI. CHẠY ỨNG DỤNG
------------------
1. Đảm bảo SQL Server đang chạy.
2. (Tùy chọn) Đảm bảo Redis đang chạy.
3. Kích hoạt venv:

    venv\Scripts\activate.bat      (CMD)


4. Chạy ứng dụng:

    python main_gui.py
